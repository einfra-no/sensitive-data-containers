## Description
Selma is a germline variant calling workflow developed at the University of Bergen. The workflow itself is based on Snakemake and all dependencies are handled by using Docker and Singularity container technology. Selma is named after the mythical Norwegian sea serpent that supposedly lives in Lake Seljord.
